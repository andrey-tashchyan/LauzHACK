# LauzHACK Chainlit Frontend

A simple Chainlit chat interface for LauzHACK.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running the Application

Run the Chainlit app:
```bash
chainlit run app.py -w
```

The `-w` flag enables watch mode, which automatically reloads the app when you make changes.

The application will be available at `http://localhost:8000`

## Features

- Simple chat interface
- Ready for integration with backend services
- Customizable theme and configuration
